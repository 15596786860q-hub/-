## 🎓 你能获得什么?

12 周后,你将掌握:

| 阶段 | 能力 |
|------|------|
| **第 1-3 周** | 读懂 JavaScript、会用开发者工具、做出简单页面 |
| **第 4-6 周** | 多页面、组件化、云开发、登录注册 |
| **第 7-9 周** | 数据库 CRUD、支付、分享、消息推送 |
| **第 10-12 周** | 性能优化、审核发布、个人小程序上线 |

**就业 / 变现方向**:
- 🎨 接单做小程序(5000-50000/单)
- 🚀 做自己的产品(MVP 验证想法)
- 💼 进公司做前端 / 全栈
- 📚 做教程博主 / 开源项目

## 💡 学习三大原则

### 1️⃣ 动手 > 阅读
看 10 小时视频 < 动手 1 小时。**每学一个 API,立刻写 demo 试**。

### 2️⃣ AI 是老师,不是保姆
AI 能解答你的问题,但**路要你自己走**。光让 AI 生成不动手,等于没学。

### 3️⃣ 小步快跑,持续上线
别追求一次做完美。**先上线一个能用的丑版本,再慢慢迭代**。
