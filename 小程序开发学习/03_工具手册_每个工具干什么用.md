## 一、开发必备工具

### 1. 微信开发者工具 ⭐⭐⭐(必装)

**它是什么**:
**官方 IDE**,小程序开发的"主战场"。写代码、预览、调试、上传一条龙。

**能干嘛**:
- 新建项目
- 实时预览代码效果(模拟器)
- 调试器(看报错、断点)
- 上传代码到微信审核
- 真机调试(扫码在手机上跑)

**装在哪**:
官网:https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
- Windows x64 / x86 版本
- Mac 版本(Intel / Apple Silicon)
- Linux(有限支持)

**第一次打开**:
1. 用微信扫码登录
2. 新建项目 → 选"小程序"
3. 填 AppID(测试可以用"测试号"免注册)
4. 选模板 "JavaScript - 基础模板"
5. 点"创建"就进入开发界面

**AI 配合**:
> "微信开发者工具怎么新建一个项目?我没 AppID 怎么办?"
> "开发者工具里编译按钮在哪?"

---

### 2. VS Code ⭐⭐⭐(辅助编辑器)

**它是什么**:
微软出的**免费代码编辑器**,全球程序员都在用。
**为什么还要装**:开发者工具自带编辑器不够好用,写代码在 VS Code,预览在开发者工具。

**装在哪**:
https://code.visualstudio.com/

**必装插件**:
- `Chinese (Simplified) Language Pack` 中文语言包
- `wechat-snippet` 微信小程序代码片段
- `minapp` 小程序标签提示
- `ESLint` 代码检查
- `Prettier` 代码格式化
- `GitLens` Git 增强

**AI 配合**:
> "VS Code 怎么装中文?"
> "我装了 minapp 插件但没提示,怎么办?"

---

### 3. Node.js ⭐⭐⭐(必装)

**它是什么**:
让 JavaScript **能在服务器跑**的运行时。
**装它的理由**:跑 npm、写云函数、装各种工具。

**装在哪**:
https://nodejs.org/
**推荐版本**:18 或 20 LTS(长期支持版)

**核心命令**:
```bash
node -v            # 查看版本(装完后测试)
npm -v             # 查看 npm 版本
npm install xxx    # 装包
npm run dev        # 跑项目脚本
```

**AI 配合**:
> "Mac 上怎么装 Node.js?"
> "npm 装包很慢,怎么换国内源?"

---

### 4. Git ⭐⭐⭐(版本管理)

**它是什么**:
**代码版本管理工具**。每次改动都自动存档,像游戏存档。

**为什么必装**:
- 改坏代码能**秒回滚**
- 多人协作必备
- 找工作必问:"会不会 Git?"

**装在哪**:
- Windows:https://git-scm.com/
- Mac:自带,或 `brew install git`

**核心命令**:
```bash
git init              # 初始化仓库
git add .             # 添加改动
git commit -m "xxx"   # 提交
git push              # 推到远程(GitHub)
git pull              # 拉最新
git status            # 看当前状态
git log               # 看历史
```

**AI 配合**:
> "我第一次用 Git,怎么把代码推到 GitHub?"
> "我 commit 错了,怎么撤销?"

---

### 5. 微信小程序账号 ⭐⭐⭐(必注册)

**它是什么**:
**小程序的身份**,没有账号发布不了。

**注册流程**:
1. 打开 https://mp.weixin.qq.com/
2. 右上角"立即注册" → 选"小程序"
3. 填邮箱(**不能是已经注册过公众号/小程序的**)
4. 邮箱验证
5. 选主体:
   - **个人**:免费,没支付能力
   - **企业**:300 元认证费,能用支付
6. 完成后在"开发管理"能看到 **AppID**

**AI 配合**:
> "注册小程序账号需要什么材料?"
> "个人主体能实现支付吗?"

---

### 6. Chrome 浏览器(调试)

**它是什么**:
**Google 浏览器**,配合云开发的 Web 端控制台、调试 H5 等。

**必装插件**:
- `Vue DevTools`(以后学 Vue 时用)
- `React DevTools`(以后学 React 时用)
- `沉浸式翻译`(英文文档翻译)

## 二、后端 / 数据工具

### 7. 微信云开发(CloudBase)⭐⭐⭐(新手首选后端)

**它是什么**:
微信提供的**一站式后端服务**,包含:
- 云函数(写 Node.js 代码)
- 云数据库(MongoDB 风格)
- 云存储(图片视频)

**为什么强推**:
- 零服务器、零域名、零备案
- 新手也能 1 天搞定后端
- 每月免费额度够个人用

**怎么开通**:
1. 开发者工具里,点顶部"云开发"按钮
2. 同意协议,开通(选"按量付费",新用户有赠送额度)
3. 创建环境,取一个环境 ID

**典型代码**:
```js
// 前端直接调
wx.cloud.callFunction({
  name: "login",
  data: { username: "xiaoming" }
}).then(res => console.log(res))
```

**AI 配合**:
> "云开发和自己买服务器有啥区别?"
> "云函数怎么写一个登录接口?"

---

### 8. Postman / Apifox(接口测试)⭐⭐

**它是什么**:
**接口调试神器**,不用开发小程序就能测后端接口。

**Postman vs Apifox**:
- Postman:国外老牌,英文界面
- Apifox:**国产,中文,推荐**
- 免费版都够用

**装在哪**:
- https://www.apifox.cn/

**典型场景**:
- 后端写好接口 → 用 Postman 测 → 小程序再对接

---

### 9. Charles / Fiddler(抓包,进阶)

**它是什么**:
**抓 HTTP/HTTPS 请求**的工具,看小程序和服务器在传什么。
**用途**:调试网络问题、学习别人小程序的接口。

**装在哪**:
- Charles:https://www.charlesproxy.com(付费,破解版多)
- Fiddler:微软,免费

**新手可以跳过**,学到中级再说。

## 三、UI / 设计工具

### 10. Figma(设计原型)⭐⭐

**它是什么**:
**在线设计工具**,画 UI 原型、切图。
**免费版够用**,不用下载客户端(浏览器就行)。

**装在哪**:
https://www.figma.com/

**替代**:
- **墨刀**:国产,中文界面
- **即时设计**:国产 Figma 平替

---

### 11. Vant Weapp ⭐⭐(UI 组件库)

**它是什么**:
**有赞开源**的小程序 UI 组件库,有 60+ 现成组件。
**为什么推荐**:
- 免费、开源、文档中文
- 大量现成组件,拿来就用
- 节省 80% UI 开发时间

**怎么用**:
1. 项目根目录跑 `npm install @vant/weapp`
2. 微信开发者工具 → 工具 → 构建 npm
3. 在 `app.json` 引入组件:
```json
{
  "usingComponents": {
    "van-button": "@vant/weapp/button/index"
  }
}
```

**官网**:https://vant-contrib.gitee.io/vant-weapp/

**替代**:
- **TDesign**:腾讯出品
- **ColorUI**:国产,颜值高
- **iView Weapp**

## 四、设备 & 真机工具

### 12. 小程序开发助手(App)⭐⭐

**它是什么**:
微信官方的**真机调试 App**,专门用来预览开发中的小程序。
**安装**:微信内搜索"小程序开发助手"

**用法**:
- 在开发者工具点"预览"生成二维码
- 用"小程序开发助手"扫码(有日志功能)

---

### 13. 手机(真机调试必备)

**用哪个**:
- 苹果 / 安卓都行
- 最好有 **1 部苹果 + 1 部安卓** 都测试
- 安卓测兼容性,苹果测性能

---

## 五、进阶工具

### 14. HBuilderX(Uni-app IDE)

**它是什么**:
DCloud 出的 **Uni-app 开发工具**,适合做跨端项目。
**跳过条件**:只做微信小程序的人不用管。

---

### 15. Taro CLI(跨端框架)

**它是什么**:
京东开源的跨端框架,**用 React/Vue 写一次,编译成多端小程序**。
**装**:`npm install -g @tarojs/cli`
**难度**:高,建议原生小程序熟练后再学。

---

### 16. 阿里云 / 腾讯云(自建服务器)

**什么时候用**:
- 云开发不够用
- 需要复杂的后端逻辑
- 企业级项目

**需要买**:
- 服务器(ECS / 云服务器):最低配 60-100 元/月
- 域名:50 元/年
- SSL 证书:免费(阿里云可申请 DV)
- 备案:免费,要 2-3 周

**新手建议**:先用云开发,**真有客户订单再上自建**。

## 💡 工具学习通用流程(配合 AI)

对于任何新工具,按这个流程学:

```
Step 1: 问 AI "XXX 工具是干什么的?一句话说明"
Step 2: 问 AI "在 [我的系统] 上怎么安装 XXX?"
Step 3: 跟着装,报错就把错误粘贴给 AI
Step 4: 问 AI "新手入门 XXX 最常用的 5 个操作是?"
Step 5: 找个小 demo 实战
Step 6: 遇到场景再深入问 AI
```
