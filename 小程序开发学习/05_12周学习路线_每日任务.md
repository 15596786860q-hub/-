# 05 | 12 周学习路线(每周目标 + 每日任务)

> **按这份计划走,12 周后你能独立开发并上线一个小程序。**
> 每天 1.5 - 2 小时,每周 1 次实战,全程用 AI 辅助。

---

## 🗓 总览:12 周 4 阶段

| 阶段 | 周次 | 目标 | 毕业标志 |
|------|-----|------|---------|
| **阶段一:编程基础** | 第 1-3 周 | HTML/CSS/JS 基础 | 能读懂 JS 代码 |
| **阶段二:小程序入门** | 第 4-6 周 | 原生小程序 | 能做多页面应用 |
| **阶段三:云开发+高级** | 第 7-9 周 | 后端+组件 | 能做有登录/数据的 App |
| **阶段四:上线+进阶** | 第 10-12 周 | 发布+优化 | 第一个上线的小程序 |

---

## 📅 第 1 周:HTML + CSS 速成

### 🎯 本周目标
- 能看懂 HTML 标签
- 能写基本的 CSS 样式
- 理解 Flex 布局(小程序必备)

### 📝 每日任务

**Day 1(1.5h)**:
- 读 00 + 01 + 07(避坑部分)
- 装好微信开发者工具 + VS Code + Node.js
- AI 提问:"HTML/CSS/JS 三兄弟分别干什么?"

**Day 2(2h)**:
- HTML 基础 10 个标签(div、span、p、h1-h6、a、img、ul、li、input、button)
- 自己写一个"个人简介"页面(.html 文件)
- AI 辅助:"给我 10 个 HTML 标签的作用和例子"

**Day 3(1.5h)**:
- CSS 基础(颜色、字体、边距、大小)
- 给昨天的简介页加样式
- 重点:盒模型(margin / padding / border)

**Day 4(2h)**:
- Flex 布局(display:flex, justify-content, align-items)
- 练习:做一个"卡片列表"布局(3 张卡片横排)
- AI 提问:"Flex 布局 10 个常用属性举例"

**Day 5(1.5h)**:
- CSS 伪类(:hover, :active)
- 响应式基础(媒体查询)

**Day 6(2h)**:
- 综合练习:模仿一个"小红书卡片"的样式
- 让 AI 给你截图参考 + 点评你的实现

**Day 7(验收)**:
- AI 出 5 道 HTML/CSS 小题
- 自测通过 > 70% 才进下一周

### ✅ 本周验收
- [ ] 装好所有工具
- [ ] 能独立写一个带样式的 HTML 页面
- [ ] 理解 Flex 布局基本用法

## 📅 第 2 周:JavaScript 基础

### 🎯 本周目标
- 掌握 JS 核心语法
- 能写简单函数和数组操作

### 📝 每日任务

**Day 4**:数组方法

**Day 5**:对象

**Day 6**:DOM 和事件
- querySelector / addEventListener
- 练习:做一个"点击按钮换颜色"的网页

**Day 7(综合)**:

## 💡 执行建议

1. **别跳步**:前面跳了,后面一定补回来
2. **每天打卡**:哪怕只做 30 分钟
3. **卡住先问 AI**,30 分钟还没解决再发帖求助
4. **周末务必做实战**,光学不练等于白学
5. **建一个笔记库**(Notion / Obsidian),把每周的知识沉淀下来
6. **加入微信开发者社区**,有人同行更容易坚持
