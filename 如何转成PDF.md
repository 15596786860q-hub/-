# 如何把服务清单转成专业 PDF

下面给你 3 种方式，从最简单到最专业，**选一个就够**。

---

## 方式 1：最简单 —— 用 Typora（推荐新手）

1. 下载 [Typora](https://typora.io/)（免费试用版够用，或找破解版）
2. 用 Typora 打开 `服务清单.md`
3. 选一个你喜欢的主题：`主题` → 推荐 `Whitey` 或 `Github`
4. `文件` → `导出` → `PDF`
5. 完成

**优点**：5 分钟搞定，样式专业
**缺点**：Typora 已经收费，但试用足够你用很久

---

## 方式 2：在线工具 —— 零安装

1. 打开 [https://md2pdf.netlify.app/](https://md2pdf.netlify.app/) 或 [https://www.markdowntopdf.com/](https://www.markdowntopdf.com/)
2. 把 `服务清单.md` 的内容复制进去
3. 选择样式，点击下载 PDF

**优点**：零安装，免费
**缺点**：样式选择有限，有些在线工具有水印

---

## 方式 3：最专业 —— 用 VS Code 插件

1. 安装 VS Code（免费）
2. 装插件：`Markdown PDF`（作者 yzane）
3. 打开 `服务清单.md`
4. 右键 → `Markdown PDF: Export (pdf)`
5. 完成

**优点**：完全免费，样式可定制
**缺点**：要装 VS Code

---

## 方式 4：美化升级（可选）

如果想要更漂亮的版式（封面、分页、专业排版），可以：

### 选项 A：给我你的信息，我帮你做封面

告诉我：
- 你想叫什么工作室名（比如"老王的工具铺"、"独立开发者-某某"）
- 有没有 Logo 或想用什么颜色（蓝色/绿色/灰色）
- 微信号、闲鱼 ID

我帮你在 Markdown 里加上封面页和页脚品牌。

### 选项 B：用 Canva 或稿定设计

PDF 导出后，用 [Canva](https://www.canva.com/) 或 [稿定设计](https://www.gaoding.com/) 的"商务画册"模板，把内容搬进去，做成带封面的版式。

---

## 发送建议

- **发客户时命名**：`技术服务报价单_2026.pdf`（比"服务清单.pdf"正式）
- **发微信**：直接发文件；不要发压缩包，客户不会解压
- **发闲鱼**：闲鱼不支持发 PDF，复制"简版文案"中对应的场景发给客户，想看详细版让他加微信
- **文件大小**：控制在 1MB 以内，太大不方便传

---

## 下一步

1. 先用方式 1 或 2 生成 PDF
2. 自己打开看一遍，找错别字、改你的联系方式
3. 挂闲鱼之前，先发给一个朋友看看，他觉得专业 = 客户也会觉得专业
4. 挂出去后，收到第一个咨询就试着发 PDF，看客户反应

**如果发送后客户不回复**：不是 PDF 的问题，是你挂品的客户质量问题（吸引来的是白嫖党）。这时候要回来改挂品文案，不是改 PDF。
