## ⚖️ 一、法律红线(务必看完)

### 🚨 绝对不能做的事

| 行为 | 法律后果 |
|------|---------|
| 破解商业软件并发布破解版 | 《著作权法》《计算机软件保护条例》,可能刑事 |
| 逆向金融类 App(银行、支付)并传播 | 《网络安全法》《刑法》 |
| 未经授权入侵他人系统 | 《刑法》285/286,非法侵入计算机信息系统罪 |
| 窃取、倒卖他人数据 | 《个人信息保护法》《刑法》,数据侵犯罪 |
| 制作销售外挂 / 破解工具 | 《刑法》,提供侵入计算机程序罪 |
| 恶意抓取大量数据(爬虫) | 《刑法》,非法获取计算机信息系统数据罪 |
| 逆向 App 绕过支付 | 诈骗罪 / 盗窃罪 |

### ✅ 可以做的事

| 行为 | 前提 |
|------|-----|
| 逆向**自己**开发的 App | 无需前提 |
| 逆向**开源** App(看 License) | 遵守 License 条款 |
| 逆向**官方靶场 / CrackMe** | 它们就是给你练的 |
| 分析自己公司授权的 App | 有书面授权 |
| 做安全研究并通过 SRC 提交漏洞 | 走官方渠道 |
| 分析自己设备上的 App 用于**学习** | 不传播、不牟利 |
| 给自己用的"去广告"修改 | 不传播、不商用 |

### 🏛 三大法律文件(了解即可)

1. **《网络安全法》**(2017 年施行)
2. **《数据安全法》**(2021 年施行)
3. **《个人信息保护法》**(2021 年施行)

**刑法相关条款**:
- 第 285 条:非法侵入计算机信息系统罪
- 第 286 条:破坏计算机信息系统罪
- 第 287 条:利用计算机实施犯罪
- 第 253 条之一:侵犯公民个人信息罪

### 💡 新手保命 3 条军规

1. **只在自己设备、自己账号上做实验**
2. **任何"研究成果"不对外传播、不商用**
3. **发现漏洞走 SRC,不黑产**

## ❓ 三、常见问题 FAQ

### 基础类

#### Q1:我是零基础文科生,真能学会吗?
**A**:能,但要更慢一些。
- 前 3 周会很痛苦(Java 语法 + 英文报错)
- 度过第 3 周,就是下坡路
- 关键是**动手 + 问 AI**

#### Q2:需要多长时间能"出师"?
**A**:
- 入门(能抓包 + 简单 Hook):**2 个月**
- 进阶(能写爬虫):**6 个月**
- 中级(能接商业单):**1 年**
- 高级(能拿 CTF 奖):**2-3 年**

#### Q3:没手机只有电脑,能学吗?
**A**:能,用模拟器起步。
- 推荐 MuMu / 雷电
- 但**第 7 周学 Frida 时强烈建议买二手 Pixel**(几百块)

#### Q4:Windows / Mac / Linux 哪个好?
**A**:
- **Mac**:最舒服(Unix 底层,工具兼容好)
- **Linux**:工具最全(Ubuntu 推荐)
- **Windows**:也能用,需要装 WSL2 模拟 Linux

#### Q5:英语不好怎么办?
**A**:不是大问题。
- 装**沉浸式翻译**插件(免费)
- 报错粘给 AI 直接翻译
- 术语记熟 100 个就够用

### 工具类

#### Q6:模拟器还是真机?
**A**:
- **入门阶段**:模拟器够了
- **Frida 开始**:真机更可靠,模拟器经常被 App 检测
- **最终建议**:二手 Pixel + Magisk 是黄金组合

#### Q7:jadx 打开 APK 特别卡 / 反编译失败?
**A**:
- 大 APK(> 100MB)确实慢,加内存参数:`jadx-gui -Xmx8g`
- 反编译失败可以用 `-r`(只反编译,不解析资源)
- 还是不行?用 **JEB Pro** 或 **IDA Pro**

#### Q8:Frida 连不上手机?
**A**:检查清单:
1. `adb devices` 能看到设备吗?
2. frida-server 跑起来了吗?(`adb shell "ps | grep frida"`)
3. 架构对吗?ARM64 手机要用 ARM64 版本的 server
4. 权限对吗?`chmod 755 frida-server`
5. 都不行 → 把报错发给 AI

#### Q9:Charles 抓不到某 App 的包?
**A**:常见原因:
1. 没装证书或装的是用户证书(Android 7+ 要系统证书)
2. App 有 SSL Pinning → 需要 Frida 绕过
3. App 用了 HTTP 代理检测 → 会不走代理
4. App 走 WebSocket / QUIC → Charles 抓不了,要用 Wireshark

#### Q10:IDA Pro 买不起,盗版能用吗?
**A**:
- 盗版在国内灰色地带,但**企业用会吃官司**
- 推荐用**Ghidra**(免费开源,NSA 出品)
- 学生可申请 IDA 教育版

### 学习类

#### Q11:一天学多久合适?
**A**:
- **每天 1-2 小时**比每周 1 次 8 小时有效 10 倍
- 关键在**持续**,别一上来就爆肝三天然后放弃 2 个月

#### Q12:卡住一个问题好几天怎么办?
**A**:按顺序尝试:
1. 去 `02_名词大辞典` 看看是不是有术语不懂
2. 问 AI,给足上下文(系统、命令、报错)
3. Google / 看雪论坛搜关键词
4. 发帖求助(看雪论坛 / 吾爱破解)
5. **跳过这个点,先往下学**,回头再看

#### Q13:AI 有时候答得不对怎么办?
**A**:
- AI 有"幻觉",不要 100% 信
- 关键步骤**自己跑一下验证**
- 不对就追问 "你刚才说的 XX 函数,我查文档没找到,能确认一下吗?"

#### Q14:要不要买课?
**A**:
- **免费资源已经够了**:看雪公开课、B 站、YouTube、官方文档
- 买课的价值在"省时间"和"有人答疑"
- 推荐(不是广告):
  - **r0ysue** 的 Frida 课(国内顶级)
  - **看雪小程序**(体系化)
  - **Z0ne Learning**(国外)

#### Q15:要不要看书?
**A**:看,但别指望看完。
- 《Android 软件安全权威指南》(非虫)—— **必看**,打基础
- 《逆向工程核心原理》—— 通用逆向思维
- 看 30% 动手比看 100% 不动手强

## 🚫 四、新手最容易踩的 10 个坑

### 坑 1:上来就挑战大厂 App
微信、淘宝、抖音有全球最强的加固团队。**先啃小 App**。

### 坑 2:工具全都装一遍
一开始只装 **jadx + apktool + ADB**。其他用到再装。

### 坑 3:盗版 IDA 被查到
学生党用 Ghidra,别碰盗版 IDA。

### 坑 4:Frida 升级后手机端没升级
电脑端 frida-tools 和手机端 frida-server 版本必须一致。

### 坑 5:改 smali 忘了改 `.locals`
**90% 的 smali 改包失败都是这个原因**。AI 能帮你检查。

### 坑 6:重打包后忘了签名
Android 拒绝安装未签名 APK。用 `apksigner sign`。

### 坑 7:主力机刷机变砖
**千万别用自己的主力机刷 Root**,买二手学习机。

### 坑 8:逆向朋友公司 App 被追责
"我没恶意的"不成立。哪怕学习,**也要挑公开可下载的 App**,别逆向内部系统。

### 坑 9:帮人写爬虫赚钱进去了
爬虫私活 = 高危行业。涉及用户数据或金融数据一律不接。

### 坑 10:学着学着变成"收藏家"
保存 500GB 视频 = 学会。**动手做 1 个案例 > 看 100 小时视频**。

## 📜 六、技术伦理自检清单

开始一个新项目前,问自己:

- [ ] 这个 App 我有合法下载 / 授权吗?
- [ ] 我的研究会不会被用于伤害用户?
- [ ] 我会不会传播未脱敏的数据?
- [ ] 我的"成果"(如脚本、工具)会不会被滥用?
- [ ] 如果被问起,我敢对家人 / 律师解释吗?

**全部打勾 → 可以做**
**任何一项打叉 → 停下来重新评估**

## 💎 七、成为一个"好"的逆向工程师

技术只是工具,关键是用它做什么。

**优秀的逆向工程师**:
- 用技术**保护**用户(发现漏洞提交厂商)
- 用技术**创造价值**(合规爬虫、安全产品)
- 用技术**教育他人**(写教程、做讲师)
- 用技术**应对对抗**(企业反作弊、反爬虫)

**糟糕的"技术人"**:
- 破解收钱
- 倒卖数据
- 帮黑产开发
- 用技术恐吓他人

**选第一种,路越走越宽;选第二种,早晚进去。**
