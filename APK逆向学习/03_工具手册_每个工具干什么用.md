## 一、静态分析工具(看代码用)

### 1. jadx / jadx-gui ⭐⭐⭐(必装)

**它是什么**:
把 APK 直接反编译成 Java 伪代码的神器。**小白的第一把武器**。

**能干嘛**:
- 拖入 APK 自动反编译
- 看到结构清晰的 Java 代码(几乎和源代码一样)
- 搜字符串、跳转引用、看类继承

**装在哪**:
- GitHub 下载:https://github.com/skylot/jadx
- 下载 `jadx-gui-xxx.zip`,解压后双击 `bin/jadx-gui.bat`(Windows)或 `bin/jadx-gui`(Mac/Linux)

**怎么用(小白 5 步)**:
1. 打开 jadx-gui
2. 文件 → 打开 APK
3. 左侧看类,双击打开
4. 按 `Ctrl+Shift+F` 搜索字符串(比如 App 里的提示文字)
5. 右键 → "查看调用"追踪函数

**AI 配合**:
> "这段 Java 代码在做什么?[粘贴代码]"
> "jadx 打开 APK 后,我在哪里能找到主界面的入口?"

---

### 2. apktool ⭐⭐⭐(必装)

**它是什么**:
老牌的 APK 解包工具,能把 APK 拆成 smali + 资源,**还能重新打包**(jadx 不行)。

**能干嘛**:
- 解出 smali 代码(可以修改)
- 解出 AndroidManifest(可读的 XML)
- 修改后重新打包成 APK

**装在哪**:
- 需要先装 JDK
- 下载:https://apktool.org/

**核心命令**:
```bash
apktool d app.apk          # 解包(d = decode)
apktool b app_folder       # 重打包(b = build)
```

**AI 配合**:
> "apktool 解包后报错 xxx,怎么办?"
> "我想把 App 里的 'VIP' 文字改成 '免费',在哪个文件里改?"

---

### 3. MT 管理器 / NP 管理器 ⭐⭐

**它是什么**:
**手机上就能用**的 APK 逆向工具,安卓界的"小瑞士军刀"。

**能干嘛**:
- 手机上直接打开 APK,修改 smali、改图标、改名字
- 傻瓜式签名重打包
- 新手最容易上手的工具

**装在哪**:
MT 管理器官网 / 应用商店(有免费版和 VIP 版)

**适合人群**:
- 只想去广告、改小东西的新手
- 没电脑只有手机的同学

---

### 4. Android Studio ⭐⭐⭐(环境必装)

**它是什么**:
Google 官方的 Android 开发 IDE。

**逆向用途**(不用来开发):
- **装 Android 模拟器**(学习前期没手机就靠它)
- 配合 smalidea 插件做 smali 断点调试
- 看 AndroidManifest 的结构化视图

**装在哪**:
https://developer.android.com/studio

---

### 5. Android Killer(老)/ AndroidCrack Tool

**它是什么**:
国产的 APK 逆向集成工具(图形化),整合了 apktool、dex2jar、jd-gui。

**现状**:被 jadx 基本取代,但老教程经常提到。新人可以跳过。

## 二、环境 & 设备工具

### 6. ADB ⭐⭐⭐(必装)

**它是什么**:
Android 调试桥,电脑和手机对话的命令行工具。

**装在哪**:
- 跟着 Android Studio 一起装(最省事)
- 或单独下 Platform-Tools:https://developer.android.com/tools/releases/platform-tools

**核心命令**:
```bash
adb devices                  # 查看连接的设备
adb install app.apk          # 安装 APK
adb uninstall 包名           # 卸载
adb shell                    # 进入手机 shell
adb logcat                   # 看 App 日志
adb push 电脑文件 手机路径   # 推文件到手机
adb pull 手机文件 电脑路径   # 从手机拉文件
```

**AI 配合**:
> "adb devices 显示 unauthorized 怎么办?"
> "怎么用 adb 快速看一个 App 的日志?"

---

### 7. Magisk ⭐⭐⭐(Root 必装)

**它是什么**:
给手机 Root 的工具,且能"隐藏 Root"让 App 检测不到。

**为什么需要 Root**:
- Frida 在 Root 环境更稳定
- 很多 Hook 操作要 Root 权限
- 才能装 LSPosed

**怎么 Root**:
1. 解锁 BootLoader(每个品牌不同,搜"[手机型号] 解锁 BL")
2. 刷入 Magisk 官方 zip
3. 装 Magisk App

**新手建议**:
- **别用主力机!** 买个二手 Pixel(Google 亲儿子最好刷机)
- 型号推荐:Pixel 3a / 4a / 5(价格 300-1000)

**AI 配合**:
> "我有一台 Pixel 5,怎么刷 Magisk 获取 root?步骤详细点"

---

### 8. 模拟器(MuMu / 雷电 / Genymotion)

**它是什么**:
电脑上跑安卓的软件,没手机时的替代方案。

**选哪个**:
- **MuMu / 雷电**:国产,稳定,有 Root 选项
- **Genymotion**:国外的,干净,付费
- **Android Studio 模拟器**:官方,配置麻烦但可靠

**注意**:
- 很多 App **检测模拟器**,会直接退出
- 学到后期要学"过模拟器检测",或直接用真机

## 三、动态分析工具(Hook 用)

### 9. Frida ⭐⭐⭐⭐⭐(逆向核心)

**它是什么**:
**动态 Hook 神器**,运行时修改 App 行为。
> ⚠️ 不夸张地说,**学会 Frida 是逆向入门和进阶的分水岭**

**能干嘛**:
- 打印任意函数的参数
- 修改函数返回值(比如让"登录失败"变"登录成功")
- 调用 App 里的私有方法
- 绕过证书锁定、Root 检测、反调试

**装在哪**:
1. 电脑端:`pip install frida-tools`
2. 手机端:下载对应架构的 `frida-server`,推到手机运行
   - 下载地址:https://github.com/frida/frida/releases

**核心概念**:
- 用 **JavaScript** 写 Hook 脚本
- 把脚本注入到目标 App 进程

**最简单的 Hook 例子**:
```javascript
Java.perform(function() {
    var MainActivity = Java.use("com.example.app.MainActivity");
    MainActivity.checkVip.implementation = function() {
        console.log("VIP 检测被调用");
        return true;  // 永远返回 true,破解 VIP
    };
});
```

**AI 配合**(最常用):
> "帮我写一个 Frida 脚本,Hook com.xxx.App 的 login 方法,打印参数"
> "Frida 报错 'Failed to spawn xxx',怎么解决?"
> "这段 smali 代码怎么翻译成 Frida Hook 脚本?"

**学习资源**:
- 官方文档:https://frida.re
- 中文教程推荐:**r0ysue 的 B 站视频**(国内最好的 Frida 讲师)

---

### 10. Objection ⭐⭐(Frida 的"自动挡")

**它是什么**:
基于 Frida 封装的**命令行工具**,很多 Hook 不用写脚本,一行命令搞定。

**装**:`pip install objection`

**神技**:
```bash
objection -g 包名 explore             # 进入 App
android sslpinning disable           # 一键绕过证书锁定
android root disable                  # 一键绕过 Root 检测
android hooking list classes          # 列出所有类
android hooking watch class_method xxx  # 监控某个方法
```

**适合**:
- 不想写脚本的场景
- 快速做常规绕过

---

### 11. Charles ⭐⭐⭐(抓包必装)

**它是什么**:
**最流行的 HTTP/HTTPS 抓包工具**,图形化,新手友好。

**装在哪**:
https://www.charlesproxy.com(有破解版,付费版 50 美金)

**基本配置**(抓 HTTPS):
1. Charles 启动 → 开启 SSL Proxying
2. 手机设置代理 → 电脑 IP:8888
3. 浏览器访问 `chls.pro/ssl` 下载证书
4. 手机装证书(用户证书 + 移到系统证书)

**注意**:
- Android 7+ 默认不信任用户证书,要装到系统级
- 遇到 SSL Pinning 要配合 Frida 绕过

**AI 配合**:
> "Charles 抓不到某个 App 的 HTTPS,显示 unknown,怎么办?"

---

### 12. mitmproxy ⭐⭐(程序员友好)

**它是什么**:
命令行抓包工具,用 **Python 脚本自动化**处理请求/响应。

**为什么选它**:
- 免费开源(Charles 收费)
- 可以用 Python 写脚本,适合做爬虫/自动化

**装**:`pip install mitmproxy`

**运行**:`mitmweb`(有网页界面,比纯终端好用)

---

### 13. LSPosed ⭐⭐(进阶)

**它是什么**:
现代版 Xposed,用于**持久化 Hook**(开机自动加载模块)。
前置:需要 Root + Magisk。

**和 Frida 的区别**:
- Frida:临时 Hook,App 关闭就失效(适合分析)
- LSPosed:开机就加载,永久生效(适合改造 App)

**适合**:
- 写一个去广告模块,自动对所有 App 生效

## 四、Native 层工具(分析 .so 文件)

### 14. IDA Pro(收费之王)

**它是什么**:
逆向界的"倚天剑",**最强的二进制分析工具**。
**价格**:商业版几万块(新手不用买,先学 Ghidra)。

**核心能力**:
- 打开 .so / .exe / .dll
- 按 **F5** 把汇编变成 C 伪代码(改变世界的一个快捷键)
- 交叉引用、调试、脚本自动化

---

### 15. Ghidra ⭐⭐⭐(免费替代 IDA)

**它是什么**:
**NSA 开源的**免费逆向工具,功能接近 IDA Pro。
**强烈推荐小白从这个开始学**,别被 IDA 盗版的法律问题困扰。

**装在哪**:
https://ghidra-sre.org(需要 Java)

**AI 配合**:
> "Ghidra 反编译出来这段代码,帮我解释一下在做什么:[粘贴]"

---

### 16. Unidbg ⭐⭐(爬虫党最爱)

**它是什么**:
用 Java **模拟执行** `.so` 文件的框架。
**神奇之处**:不用手机、不用模拟器,直接在电脑上跑 so。

**典型场景**:
App 的 sign 签名算法藏在 so 里,你用 Unidbg 把 so 加载起来,传参调用,就能批量生成签名(爬虫必备)。

**装**:
https://github.com/zhkl0228/unidbg(Java 项目,用 IDEA 打开)

---

### 17. radare2 / Cutter

**它是什么**:
**免费开源**的逆向工具全家桶。
- `radare2`:命令行(geek 向)
- `Cutter`:radare2 的 GUI(推荐新手用这个)

## 五、脱壳 / 加固对抗工具(进阶)

### 18. FART

**它是什么**:
**脱壳机**,刷到手机里,能把加固 App 的原始 DEX "挤"出来。
**原理**:在 ART 层 Hook,主动调用每个方法,让代码被加载到内存,然后 dump。

---

### 19. BlackDex / FRIDA-DEXDump / Youpk

**用途**:都是脱壳工具,适用不同场景。
- **BlackDex**:免 Root 脱一些弱加固
- **FRIDA-DEXDump**:基于 Frida 的脱壳
- **Youpk**:全量脱壳机

## 💡 工具学习通用流程(配合 AI)

对于任何新工具,按这个流程学:

```
Step 1: 问 AI "XXX 工具是干什么的?一句话说明"
Step 2: 问 AI "在 [我的系统] 上怎么安装 XXX?"
Step 3: 跟着装,报错就把错误粘贴给 AI
Step 4: 问 AI "新手入门 XXX 最常用的 5 个命令/操作是?"
Step 5: 找个 CrackMe App 实战
Step 6: 遇到场景再深入问 AI
```
